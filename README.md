# FloppaLand Modpack

Minecraft сборка построенная вокруг Create Aeronautics, построенная на базе **NeoForge 1.21.1**. Проект ориентирован на автоматизацию, глубокую оптимизацию и создание кастомного игрового опыта взаимодействия между игроками.


## Установка клиента

1. Откройте MultiMC лаунчер (или любой совместимый, например, Prism Launcher / PolyMC).
2. Выберите импорт сборки по ссылке.
3. Вставьте прямую ссылку на манифест сборки:
   ```text
   https://raw.githubusercontent.com/Zizazar/FloppaLand-modpack/main/pack.toml
   ```
